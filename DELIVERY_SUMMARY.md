## How to Get Started

### Quick Start (5 minutes)

1. **Open SETUP.md** and follow Steps 1-3:
   - Create bot with @BotFather
   - Get your admin chat ID
   - Update TOKEN and ADMIN_CHAT_ID in `str8tv_bot.py`

2. **Test locally:**
   ```bash
   pip install -r requirements.txt
   python3 str8tv_bot.py
   ```

3. **Send `/start` to your bot** on Telegram

### Deploy (10 minutes)

4. **Open DEPLOYMENT.md** and choose:
   - Railway (easiest, free tier available)

5. Follow the deployment guide step-by-step

### Manage Ongoing

6. **Use USER_GUIDE.md** for:
   - Updating FAQs

   - Customizing responses

## What You Need to Do

Before deploying, you need to:

1. **Get Telegram Bot Token** (SETUP.md - Step 1)
   - Talk to @BotFather
   - Create your bot
   - Copy the token

2. **Get Your Admin Chat ID** (SETUP.md - Step 2)
   - Use @userinfobot OR
   - Check bot logs after first message

3. **Update Configuration** (SETUP.md - Step 3)
   - Open `str8tv_bot.py`
   - Replace `TOKEN` on line 11
   - Replace `ADMIN_CHAT_ID` on line 12

4. **Add Video Links** (SETUP.md - Step 4)
   - Replace placeholder YouTube links (lines 76-82)
   - Use your actual tutorial videos
